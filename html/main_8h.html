Função principal do sistema de gestão de vacinas.

Parameters
argcnúmero de argumentos da linha de comandos
argvarray de argumentos da linha de comandos
Returns
int 0

Inicializa a estrutura do sistema de vacinas com valores padrão:

  • Listas vazias para vacinas, utentes e inoculações
  • Contadores a zero
  • Data atual definida para 1-1-2025

Processa o idioma conforme o argumentos recebido e entra num loop que lê e processa comandos até encontrar EOF.

Note
A data inicial é alterada através do comando 't'

Parameters

Analisa o comando recebido e chama a função correspondente para o processar. No caso do comando 'q', liberta a memória e termina o programa. Para comandos desconhecidos, limpa o buffer de entrada.

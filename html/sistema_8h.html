Vacina * melhor_opcao )

Cria e regista uma nova inoculação no sistema.

Parameters
sistemaPonteiro para o sistema
utentePonteiro para o utente vacinado
melhor_opcaoPonteiro para a vacina administrada

Esta função realiza todas as operações necessárias para registar uma nova inoculação:

  1. Cria a estrutura de inoculação
  2. Adiciona à lista do utente
  3. Adiciona à lista global do sistema
  4. Atualiza os contadores da vacina
Note
A função trata de toda a gestão de memória necessária.

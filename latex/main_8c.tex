\doxysubsection{Detailed Description}
Ficheiro principal do sistema de gestão de vacinas. 

Este ficheiro contém a função main do sistema de vacinas e o processamento dos comandos. Implementa o loop principal do programa e a distribuição dos comandos para as respetivas funções de processamento.
